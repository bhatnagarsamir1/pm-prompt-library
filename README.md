## How to Use

Each prompt follows a consistent format:

- **Title** — what the prompt does
- **Context** — when and why to use it
- **Prompt** — the actual text, ready to copy and adapt
- **Variables** — placeholders in `[brackets]` to customize

Copy the prompt, replace the variables with your specific 
context, and paste into your AI tool of choice.

## Contributing

This library grows through use and iteration. If you have a 
prompt that has saved you time or improved your output quality:

1. Fork the repo
2. Add your prompt to the relevant folder using the format above
3. Submit a pull request with a brief note on what it does

Keep prompts practical, specific, and tested. No theoretical 
prompts — only ones you've actually used.
